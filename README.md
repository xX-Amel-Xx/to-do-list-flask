# To-Do List com Flask

Projeto desenvolvido para praticar desenvolvimento web com Flask, HTML e CSS, implementando um CRUD para gerenciamento de tarefas.

# Sobre o projeto

Esta aplicação permite cadastrar tarefas, marcá-las como concluídas, visualizar tarefas pendentes e excluir tarefas cadastradas.

O objetivo do projeto foi colocar em prática conceitos de desenvolvimento backend utilizando Flask e banco de dados SQLite.

---

## Tecnologias utilizadas

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML5
- CSS3

---

##  Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Visualizar tarefas pendentes
- Visualizar tarefas concluídas
- Excluir tarefas
